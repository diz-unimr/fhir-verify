# fhir-verify

> FHIR 🔥 Verification server client

## Usage

```
Usage: fhir-verify [OPTIONS] <FILENAME>

Arguments:
  <FILENAME>

Options:
      --url <URL>                    [default: https://validator.fhir.diz.uni-marburg.de/validateResource]
      --show-details
      --count
      --min-severity <MIN_SEVERITY>  [possible values: error, warning, information]
  -h, --help                         Print help
  -V, --version                      Print version
```
